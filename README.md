# Pop cat sing a song

Frontend repository - current

Backend repository - [link](https://github.com/YanaTrifonova/pop_cat_server)

## Stack

### Frontend

- JavaScript
- React
- Redux
- Axios
- Material-UI

### Backend

- Node.js
- Express.js
- JWT & BCrypt Authentication
- PostgreSQL
- Sequelize ORM
